Detailed Description

Aggregates calls from different runs of tested program into one graph.

Every time a message with call arrives from tested program, aggregator is called and checks whether such call has already arrived and in case not, it adds it into the graph. Also is able to represent graph in various formats for output. The graph structure can be modified by object of JumpDetector class which is therefore set as friend class.
